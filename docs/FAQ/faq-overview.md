# Часто задаваемые вопросы (FAQ)

## О структуре FAQ

В этой папке собраны ответы на часто задаваемые вопросы по проекту Lifelong Learning Assistant. Каждый ответ хранится в отдельном markdown файле для удобной навигации и редактирования.

### Как добавить новый FAQ

1. Создайте новый файл в этой папке с именем в формате `XX-краткое-описание.md`
2. Включите в начало файла заголовок с полным названием вопроса
3. Структурируйте ответ с помощью заголовков разных уровней
4. Добавьте ссылку в содержание данного README файла

### Рекомендации по оформлению

- Используйте чёткие и информативные заголовки
- Структурируйте ответы с помощью списков и подзаголовков
- Включайте примеры кода там, где это уместно
- Добавляйте дату создания в конце ответа
